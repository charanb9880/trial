About Huffman Coding

Huffman Coding is a lossless data compression algorithm developed by David A. Huffman in 1952. It assigns variable-length codes to input characters, with shorter codes for more frequent characters.

Algorithm Steps


1. Calculate frequency of each character in the input.
2. Create a min-heap of nodes, each storing a character and its frequency.
3. While heap has more than one node:
   a. Extract two nodes with minimum frequency.
   b. Create a new internal node with frequency equal to the sum.
   c. Assign left and right children to the new node.
   d. Insert the new node back into the heap.
4. Traverse the Huffman tree to assign codes to characters.
